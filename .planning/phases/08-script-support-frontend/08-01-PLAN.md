# Plan 08-01: Script Support & Monaco Integration

Este plan habilita la creación de nodos de script y proporciona un editor de código profesional en la UI.

## Goals
1. Añadir la herramienta "JS Script" al catálogo mediante una migración.
2. Instalar e integrar Monaco Editor en el frontend.
3. Permitir la edición de código JS en el panel de propiedades del nodo.

## Tasks

### 1. Backend: JS Script Tool
- Crear la migración `backend/alembic/versions/010_add_js_script_tool.py`.
- Insertar la herramienta `js_script` en la tabla `editor_tools`.
- Definir `prop_defs` con un campo `code` de tipo `code`.

### 2. Frontend: Dependency Setup
- Instalar `@guolao/vue-monaco-editor` en `dashboard-app`.
- Configurar el editor (temas, lenguajes) en un nuevo componente.

### 3. Frontend: CodeEditor Component
- Crear `dashboard-app/src/components/editor/CodeEditor.vue`.
- El componente debe envolver el Monaco Editor y emitir cambios al padre.
- Soportar el lenguaje `javascript`.

### 4. Frontend: FlowEditorCanvas Integration
- Importar `CodeEditor` en `FlowEditorCanvas.vue`.
- Modificar el bucle de renderizado de propiedades para detectar `type === 'code'`.
- Renderizar el `CodeEditor` en lugar de un `input` o `textarea`.

## Verification
1. Ejecutar la migración y verificar que "JS Script" aparece en la paleta de herramientas.
2. Arrastrar un nodo de script y verificar que el editor de Monaco se carga en el panel derecho.
3. Escribir código, guardar el diagrama y verificar que el código persiste al recargar.
