```javascript
function inferType(val) {
  if (val === null) return 'Null'
  if (Array.isArray(val)) return 'Array'
  const t = typeof val
  return t.charAt(0).toUpperCase() + t.slice(1)
}
```

Create a new Vue 3 SFC at `dashboard-app/src/components/editor/NodeInspectorPanel.vue` with `<script setup>`, no TypeScript, `<style scoped>`.

**Props** (as defined in interfaces above):
- `nodeId` (String, required)
- `nodeName` (String, default '')
- `nodeData` (Object, default null) — shape: `{ input, output, status, duration, start_time, end_time, logs: [] }`

**Internal state (refs):**
- `innerTab` ref defaulting to `'output'` (Salida is default per CONTEXT.md)
- `jsonExpanded` ref defaulting to `true`

**Computed: `activePayload`**
- Returns `nodeData.input` when `innerTab === 'input'`
- Returns `nodeData.output` when `innerTab === 'output'`
- Returns null otherwise

**Computed: `isTruncated`**
- `JSON.stringify(activePayload).length > 2000`

**Computed: `hasVariables`**
- `activePayload?.variables && Object.keys(activePayload.variables).length > 0`

**Computed: `statusIcon`**
- `'check_circle'` when `nodeData.status === 'success'`
- `'cancel'` when `nodeData.status === 'error'`
- `'sync'` otherwise (running)

**Functions:**
- `inferType(val)` — as defined in interfaces
- `formatVarValue(val)` — if typeof val is 'object', return `JSON.stringify(val)` (single line); else return String(val)
- `formatDuration(ms)` — if ms < 1000 return `${ms}ms`; else return `${(ms/1000).toFixed(1)}s`
- `formatTime(ts)` — `new Date(ts).toLocaleTimeString()` or `'—'` if falsy
- `displayJson(data)` — if `!jsonExpanded.value` return `'{...}'`; else return `JSON.stringify(data, null, 2) || '{}'`
- `exportData()` — build payload `{ node_id: nodeId, node_name: nodeName, status, duration, input, output, logs }` from nodeData, create Blob with JSON.stringify(payload, null, 2), URL.createObjectURL, anchor click, URL.revokeObjectURL. Filename: `node-${nodeId}-execution.json`.

**Template structure:**
```
div.nip-root (if nodeData is null: show "Sin datos de ejecución")
  div.nip-header
    div.nip-node-name  → nodeName
    div.nip-status-row
      span.nip-chip :class="nip-chip--{nodeData.status}"
        span.msi → statusIcon
        " " + nodeData.status
      span.nip-duration → formatDuration(nodeData.duration)
      button.nip-export-btn @click="exportData"
        span.msi "download"
    div.nip-timestamps
      span "Inicio: " + formatTime(nodeData.start_time)
      span "Fin: " + formatTime(nodeData.end_time)

  div.nip-tabs
    button.nip-tab :class="{active: innerTab==='output'}" @click="innerTab='output'" → "Salida"
    button.nip-tab :class="{active: innerTab==='input'}"  @click="innerTab='input'"  → "Entrada"
    button.nip-tab :class="{active: innerTab==='logs'}"   @click="innerTab='logs'"   → "Logs"

  div.nip-body
    template v-if="innerTab === 'input' || innerTab === 'output'"
      div.nip-section-hdr
        span.nip-section-title "data_object"
        button.nip-expand-btn @click="jsonExpanded = !jsonExpanded"
          → jsonExpanded ? "Contraer todo" : "Expandir todo"
      div.nip-truncated-warn v-if="isTruncated"
        "Payload truncado (demasiado grande)"
      pre.nip-json {{ displayJson(activePayload) }}

      div.nip-section-hdr (margin-top: 16px)
        span.nip-section-title "variables"
      table.nip-vars-table v-if="hasVariables"
        thead: tr: th Nombre / th Valor / th Tipo
        tbody: tr v-for="(val, key) in activePayload?.variables"
          td {{ key }} / td {{ formatVarValue(val) }} / td {{ inferType(val) }}
      div.nip-empty v-else "Sin variables"

    template v-else-if="innerTab === 'logs'"
      div.nip-empty v-if="!nodeData.logs?.length" "Sin actividad registrada"
      div.nip-log-line v-for="(log, i) in nodeData.logs" :key="i"
        :class="`nip-log--${log.severity || 'info'}`"
        span.nip-log-ts {{ log.timestamp }}
        span.nip-log-msg {{ log.message }}
```
