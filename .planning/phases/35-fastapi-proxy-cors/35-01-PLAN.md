# Plan: 35-01-PLAN - FastAPI Proxy Implementation

## Goal
Implement the proxy for FastAPI at `/bff/api/*` in the BFF, including token injection and header management.

## Proposed Changes

### 1. `bff/src/config.js`
- Add `backendUrl` (internal Docker URL for FastAPI).

### 2. `bff/src/proxy.js` (NEW)
- Implement `fastapiProxy` using `http-proxy-middleware`.
- Configure `onProxyReq` to inject Bearer token.
- Configure `onProxyRes` to strip CORS headers from the backend.

### 3. `bff/src/index.js`
- Mount `fastapiProxy` at `/bff/api`.
- Apply `requireAuth` and `tokenRefresh` middleware to the proxy route.

## Step-by-Step Execution

### Step 1: Update `config.js`
- Default to `http://backend:8000` (internal Docker name).

### Step 2: Create `bff/src/proxy.js`
- Implement the proxy logic.

### Step 3: Mount in `index.js`
- Ensure it comes after session and before any broad error handlers.
