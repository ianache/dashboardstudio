# Plan 17-01: Frontend - Connections UI

## Goals
1. Integrar el acceso a "Conexiones" en el `SideMenu`.
2. Crear la vista `ConnectionsView` para listar y gestionar fuentes.
3. Crear el modal `ConnectionEditModal` con formularios dinámicos según el tipo de conexión.

## Tasks

### 1. Navigation
- Modificar `dashboard-app/src/components/layout/SideMenu.vue` para incluir "Conexiones" bajo "DATA INTEGRATION".
- Actualizar `dashboard-app/src/router/index.js` para registrar la nueva ruta `/integrations/connections`.

### 2. Connections View
- Crear `dashboard-app/src/views/ConnectionsView.vue`.
- Listar conexiones existentes (con botón de editar/eliminar).
- Añadir botón "Nueva Conexión".

### 3. Connection Edit Modal
- Crear `dashboard-app/src/components/connections/ConnectionEditModal.vue`.
- Implementar formularios dinámicos:
    - Base: nombre, tipo (select), configuración JSON.
    - Dinámico: campos específicos para SMTP, DB, HTTP, etc.
- Integrar llamado a `/api/v1/data-sources/{id}/test`.

## Verification
1. Verificar que el enlace en SideMenu navega correctamente.
2. Validar que el modal se abre con los campos correctos según el tipo.
3. Confirmar que el botón "Test Connection" dispara el endpoint backend y muestra el resultado en la UI.
