  <files>backend/app/runtime/runner.ts</files>
  <action>
Modify `backend/app/runtime/runner.ts`:
1. **Schema**: Update `FlowConnection` interface to include `fromHandle?: string`.
2. **Topological Sort**: Check `order.length === nodes.length`. If not, throw `Error("Cycle detected")`.
3. **Node State**: Track `nodeResults` (Map of node ID to boolean result for branches).
4. **Execution Loop**:
   - For each node, check if it has **Active Inputs**. 
   - A connection is active if upstream succeeded AND (upstream is not a branch OR handle matches result).
   - If no active inputs (and not a root), add to `skippedNodes` and continue.
   - For `conditional_branch` nodes: Evaluate `expression` and store in `nodeResults`.
  </action>
