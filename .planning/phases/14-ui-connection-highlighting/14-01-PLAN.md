# Plan 14-01: UI Connection Highlighting

Este plan implementa la visualización de la secuencia de ejecución coloreando las conexiones en verde a medida que se activan.

## Goals
1. Identificar conexiones activas basándose en el estado de los nodos origen.
2. Colorear las conexiones en verde cuando el nodo de origen haya terminado con éxito.
3. Asegurar que los marcadores de flecha también se vuelven verdes.

## Tasks

### 1. Connection Highlighting Logic
- En `FlowEditorCanvas.vue`, en el bucle de renderizado de conexiones (`v-for="conn in connections"`):
    - Añadir una clase dinámica `:class="{ 'fec-conn--active': nodeExecStatus[conn.from] === 'success' }"`.
    - La clase `fec-conn--active` debe aplicarse a la etiqueta `<path>`.

### 2. SVG Markers for Active State
- En el bloque `<defs>` del SVG en `FlowEditorCanvas.vue`:
    - Añadir un nuevo marcador `fec-arr-active` con `fill="#22c55e"`.
- Modificar el atributo `:marker-end` de las conexiones:
    - `:marker-end="selectedConn === conn.id || hoveredConn === conn.id ? 'url(#fec-arr-sel)' : (nodeExecStatus[conn.from] === 'success' ? 'url(#fec-arr-active)' : 'url(#fec-arr)')"`.

### 3. Connection CSS
- Añadir CSS para la clase `.fec-conn--active`:
    - `stroke: #22c55e !important;`
    - `stroke-width: 3 !important;`
    - `transition: stroke 0.3s, stroke-width 0.3s;`

## Verification
1. Ejecutar un flujo con múltiples nodos conectados.
2. Verificar que las flechas se vuelven verdes inmediatamente después de que el nodo origen correspondiente termine con éxito.
3. Verificar que al reiniciar la ejecución, las flechas vuelven a su color gris original.
