<objective>
Refactor visible secondary components to remove hardcoded hex colors and align with the Material Design 3 theme variables and minimalist aesthetic.
</objective>

<context>
Following the migration of the core shell and main views, several secondary components (modals, cards, panels) still contain hardcoded hex colors or outdated design tokens.
We need to map these to the established CSS variables:
- `#fff`, `#ffffff` -> `var(--surface)` or `var(--on-primary)`
- `#000`, `#000000` -> `var(--on-surface)`
- `#f7f9fb`, `#f1f5f9` -> `var(--background)` or `var(--surface-container-low)`
- `#e2e8f0`, `#cbd5e1` -> `var(--border)`
- Tints/Transparencies -> `color-mix(in srgb, var(--token) N%, transparent)`
</context>
