<objective>
Implement a light/dark theme toggle system.

Purpose: Allow users to switch between the current dark design system and a new light design system (based on design/DESIGN-light.md). The selected theme persists across sessions via localStorage.
Output: Two icon buttons (sun = light, moon = dark) in the SideMenu bottom section. Switching themes applies `data-theme="light"` or `data-theme="dark"` to `<html>`, overriding CSS custom properties globally. No component changes required beyond SideMenu.
</objective>

<task type="auto">
  <name>Task 2: Add theme state and actions to uiStore</name>
  <files>dashboard-app/src/stores/ui.js</files>
  <action>
    1. Add `theme: localStorage.getItem('ui-theme') || 'dark'` to state.
    2. Add `initTheme()` action:
       ```js
       initTheme() {
         document.documentElement.setAttribute('data-theme', this.theme)
       }
       ```
    3. Add `setTheme(t)` action:
       ```js
       setTheme(t) {
         this.theme = t
         localStorage.setItem('ui-theme', t)
         document.documentElement.setAttribute('data-theme', t)
       }
       ```
    No watcher needed — actions update DOM directly.
  </action>
  <verify>
    <auto>grep "initTheme\|setTheme\|ui-theme" dashboard-app/src/stores/ui.js | wc -l — must be >= 3</auto>
