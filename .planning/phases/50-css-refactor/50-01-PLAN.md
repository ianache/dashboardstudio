# Plan 50-01: Legacy CSS Token Migration

Migrate all legacy `--text` and `--text-secondary` tokens to their MD3 equivalents and replace hardcoded white backgrounds with `var(--surface)`.

## Tasks

- [ ] Task 1: Replace `var(--text)` with `var(--on-surface)` project-wide in `dashboard-app/src` `type=auto`
- [ ] Task 2: Replace `var(--text-secondary)` and `var(--text-secondary, #666)` with `var(--on-surface-variant)` project-wide in `dashboard-app/src` `type=auto`
- [ ] Task 3: Replace hardcoded white backgrounds (`#fff`, `#ffffff`) with `var(--surface)` in `dashboard-app/src` components `type=auto`
- [ ] Task 4: Verify migration and check for any remaining legacy tokens `type=auto`
