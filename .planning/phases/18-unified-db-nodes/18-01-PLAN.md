# Plan 18-01: Refactor DB Nodes to Generic SQL

Unificar los nodos específicos de base de datos en nodos genéricos `sql_source` y `sql_destination` que utilicen las conexiones gestionadas centralmente.

## Goals
1. Crear nodos `sql_source` y `sql_destination` en el editor.
2. Implementar lógica en `runner.ts` para ejecutar queries usando `connection_id` seleccionado.
3. Crear `source_executor.py` para manejar la conexión SQLAlchemy dinámica.

## Tasks

### 1. Backend Service (`source_executor.py`)
- Crear un servicio que reciba un `connection_id` y una query.
- Usar SQLAlchemy para construir el engine dinámicamente (`create_engine`).
- Ejecutar y retornar resultados.

### 2. Deno Runner Refactor
- Actualizar `runner.ts` para añadir manejadores `sql_source` y `sql_destination`.
- El nodo debe recibir `connection_id` y `query`.

### 3. Verification
- Crear flujo con [SQL Source] -> [JS Script] -> [SQL Destination].
- Validar conexión a PostgreSQL y MySQL usando las conexiones centralizadas.
