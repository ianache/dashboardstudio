# Plan: 37-02-PLAN - Final Cleanup and App Initialization

Remove the `keycloak-js` library, refactor the application entry point, and clean up residual auth logic.

## Tasks

<task id="37-02-01">
  <name>Refactor App Entry (main.js)</name>
  <files>["dashboard-app/src/main.js"]</files>
  <action>
    - Remove all keycloak-js imports and initialization logic.
    - Remove sessionStorage sync logic and token expiry listeners.

    - Call authStore.initialize() and wait for it (async) before mounting the app to avoid UI flickering.
    - Mount the app normally.
  </action>
  <verify>

  </verify>
  <done>App entry point migrated to BFF initialization flow.</done>
</task>

  <files>["dashboard-app/src/router/index.js"]</files>
