# Plan 20-01: Execution Dashboard UI

Implementar la visualización del historial de ejecuciones y el detalle de logs por nodo.

## Goals
1. Añadir botón/icono "Log" en la lista de integraciones (`IntegrationsView`).
2. Crear modal `ExecutionHistoryModal` con tabla paginable de ejecuciones.
3. Mostrar detalle de logs (input/output) por cada nodo al seleccionar una ejecución.

## Tasks

### 1. IntegrationsView Update
- Modificar `IntegrationsView.vue` para añadir una columna o botón de "Logs" en la tabla de flujos.
- Gestionar la apertura del modal al hacer clic.

### 2. ExecutionHistoryModal
- Crear `dashboard-app/src/components/executions/ExecutionHistoryModal.vue`.
- Usar un componente de tabla para listar ejecuciones.
- Añadir paginación (limit/offset).
- Implementar panel de detalle (acordeón o modal secundario) para visualizar `node_logs` con inputs/outputs.

### 3. Verification
- Abrir historial para un flujo con ejecuciones pasadas.
- Validar paginación y carga de datos.
- Inspeccionar el detalle de un nodo con logs detallados.
