# Feat 04: Panel lateral para añadir tablas


# Requerimientos

- Mejoremos como "Añadir tabla" cambiando del botón "Añadir tabla" a un panel lateral a la izquierda que sea plegable hacia la derecha y replegable para ocultarlo. 
- Se debe permitir drag & drop de las tablas de hechos y dimensiones sobre el canvas de diseño visual. 
- Se debe permitir visualmente colocar la misma tabla (hechos y dimensiones) mas de una vez. 
- Permitir en el panel lateral filtrar las tablas por su nombre y agrupar alfabeticamente las tablas de hechos y luego el grupo de las tablas de dimensiones también ordenadas alfabeticamente. 
- Cada grupo debemos poder colapsarlo o expandirlo.
