# Goal

Rediseñar el dashboard studio para que se vea mas profesional y moderno.

# UX/UI

- Se debe mantener la misma funcionalidad que tiene actualmente el dashboard studio, pero con un UX/UI mas moderno y profesional.
- El UX/UI debe basarse en este diseño: https://stitch.withgoogle.com/projects/6261228489183515491?node-id=20f250265c9f424296b88c99f414685c
- Se debe usar un diseño minimalista y elegante, con colores suaves y neutros.
- Se deben usar iconos de alta calidad.


# TECHNOLOGIES

- El stack tecnológico debe ser el siguiente:
    - Vue.js 3.x
    - TypeScript
    - Tailwind CSS
    - Pinia
